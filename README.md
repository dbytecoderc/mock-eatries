### Installation

Follow these steps to set up the app.

#### Clone the repo:

`$ git clone https://github.com/dbytecoderc/mock-eatries.git`

#### Navigate to the project directory:

`$ cd mock-eatries`

#### Install dependencies

`npm install`

#### Start the application

`npm run start:dev`
